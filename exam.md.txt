1. 지필시험 연결 포인트 (핵심 용어 단답형 완벽 대비)

14차시 최종 지필평가의 1번 문항(핵심 용어 단답형, 배점 20점)에 대비하여 아래 7대 용어의 **핵심 지문 설명 패턴**을 꼭 외워두셔야 합니다.

| 단답형 출제 문제 키워드 (지문 예시) | 정답 용어 (한글/영문 모두 인정) |  |
| --- | --- | --- |
| 데이터센터 내부의 하드웨어 부품 교체 및 케이블 포지션 조율, 냉방 온도를 유지 보수하는 현장 운영 직무 | **DCO** (Data Center Operations) |  |
| 수십 대의 서버 장비를 한 공간에 수직 수납하여 고정하기 위해 조립된 캐비닛 형태의 철제 표준 프레임 | **Rack** (랙) |  |
| 네트워크를 통하여 고객에게 웹 서비스나 DB 자원을 공급하며 랙 슬롯에 장착되는 사각 본체 컴퓨터 장비 | **Server** (서버) |  |
| 장애 복구나 장비 교체 등 작업자가 해결해야 할 일감과 위치를 명시한 일종의 가상 업무 수행 지시서 | **Ticket** (티켓) |  |
| 데이터센터 운영 중 장애 인시던트 해결에 주어지는 시간 제약 등 품질 수준을 약속하는 고객간 서비스 계약서 | **SLA** (Service Level Agreement) |  |
| 작업 오류와 휴먼 에러를 방지하기 위해 각 점검 행위의 순서를 꼼꼼하게 규정한 표준 운영 절차 지침서 | **SOP** (Standard Operating Procedure) |  |
| 정해진 시간(SLA) 내 복구가 어렵거나 L1 범위를 초과할 때 상위 기술 엔지니어 팀에게 문제 해결을 이관하는 행위 | **Escalation** (에스컬레이션) |  |

## 16-3. 장애 원인과 확인 순서 표 작성 연습

아래 표는 지필시험에서 손으로 작성할 수 있어야 하는 구조입니다.

| 장애 표현 | 가능한 원인 분류 | 교육용 확인 순서 | 기록할 내용 |
| --- | --- | --- | --- |
| Link Down | 연결 상태 문제, 설정 변경, 장비 상태 변화 가능성 | 1. 발생 시간 확인 → 2. 관련 로그 확인 → 3. 반복 여부 확인 → 4. 영향 범위 정리 | 발생 시간, 관련 장비명, 연결 상태, 반복 여부 |
| CRC error | 데이터 전송 오류, 케이블 품질, 포트 상태, 설정 문제 가능성 | 1. 오류 증가 추세 확인 → 2. 같은 장비 반복 여부 확인 → 3. 다른 이벤트와의 시간 관계 확인 → 4. 보고 필요 여부 정리 | 오류 횟수, 반복 구간, 관련 장비, 추정이 아닌 관찰 내용 |
| SSD failure | 저장장치 상태 이상, 경고 누적 가능성 | 1. 경고 발생 시간 확인 → 2. 반복 여부 확인 → 3. 관련 티켓 여부 확인 → 4. 보고서 초안 작성 | 장비명, 경고 시간, 반복 여부, 다음 확인 필요 사항 |
| Fan Alert | 냉각 팬 상태 이상, 온도 경고와 연관 가능성 | 1. 경고 발생 시간 확인 → 2. 온도 관련 이벤트 확인 → 3. 반복 여부 확인 → 4. 전달 필요 여부 정리 | 경고 내용, 발생 시간, 관련 이벤트, 전달 대상 |

> 이 표는 교육용 시나리오 분석 연습입니다. 실제 장비나 실제 네트워크를 점검하지 않습니다.

| Link Down | 절차, 확인항목, Escalation 기준 | 1. 발생 시간 확인 → 2. 샘플 장비명 확인 → 3. 반복 여부 확인 → 4. Ticket 표현 확인 → 5. Escalation 필요 여부 정리 | 발생 시간, 샘플 장비명, 반복 여부, 관련 Ticket, 전달 필요 여부 |
| CRC error | 절차, 확인항목, 원인 단정 금지 | 1. 오류 발생 시간 확인 → 2. 증가 추세 확인 → 3. 같은 장비 반복 여부 확인 → 4. 관련 이벤트 확인 → 5. 보고 필요 여부 정리 | 오류 횟수, 반복 구간, 관련 장비, 관찰 내용 |
| SSD failure warning | 주의사항, 전달 기준 | 1. 경고 시간 확인 → 2. 반복 여부 확인 → 3. Ticket 상태 확인 → 4. Handover Memo에 남길 내용 정리 | 경고 시간, 반복 여부, 샘플 장비명, 후속 확인 필요 사항 |
| Fan Alert | 주의사항, 확인항목 | 1. 경고 시간 확인 → 2. Temperature warning 동시 발생 여부 확인 → 3. 반복 여부 확인 → 4. 전달 필요 여부 정리 | 경고 시간, 관련 이벤트, 반복 여부, 전달 필요 여부 |

> 이 표는 교육용 시나리오 분석 연습입니다. 실제 장비나 실제 네트워크를 점검하지 않습니다.
